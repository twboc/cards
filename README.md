## About the project

I wanted to show what a highly optimized version of a `react-native-skia` demo could look like. Many examples online use `useState` to drive shader timing instead of `useClock`, and a lot of them suffer from unnecessary component re-renders.

### OPTIMISATION

I wanted to do a deep dive on react native optimisation. This project implements many techniques that allow us to render multiple comonenents with animations without a stutter.

### Shaders

Skia makes it possible to pass custom shaders, which are used here mainly for the background effects. I implemented several popular shaders inspired by ShaderToy, along with a few custom ones of my own. Shader rendering optimization was a major focus.

### Gesture

I used `react-native-sensors` to provide phone movement feedback. Gestures are also supported, so the cards can be moved interactively.

### No Expo

I tried to keep dependencies to a minimum. A few packages were necessary, such as `react-native-reanimated` and `react-native-worklets`, but only where they provided clear performance benefits.
